# Simple HTML & CSS Website

## Description
This is a simple website created using **HTML** and **CSS**. In this project, I added an image, applied a background color to the body, created navigation links, and styled the links with background colors. I also added a paragraph and used the `::first-letter` CSS property to make the first letter uppercase and more attractive. This project helped me practice the basic concepts of HTML and CSS.

## Features
- Added an image
- Applied a background color to the body
- Created navigation links
- Styled links with background colors
- Added a paragraph
- Capitalized the first letter of the paragraph using CSS
- Built using HTML and CSS

## Technologies Used
- HTML5
- CSS3
